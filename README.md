# 🛒 MercadoList — Lista de Compras Inteligente

Aplicativo de lista de compras inteligente desenvolvido para facilitar o dia a dia, permitindo organização, controle de gastos e instalação no celular como um app.

---

## 🔗 Acesse o projeto online

👉 https://app-mercado-list.vercel.app/

---

## 📥 Como rodar o projeto localmente

### 1. Clone o repositório

```bash
git clone https://github.com/MichaelRc2/app-MercadoList
```

---

### 2. Acesse a pasta do projeto

```bash
cd nome-do-repositorio
```

---

### 3. Rodar o projeto

Você pode abrir o arquivo `index.html` diretamente no navegador.

Ou utilizar uma extensão como Live Server no VS Code para uma melhor experiência.

---

## 📲 Como usar o app

### 🌐 No navegador

Basta acessar o link e utilizar normalmente.

### 📱 Instalar no celular

#### Android

* Acesse o link
* Clique em "Instalar app"

#### iPhone (Safari)

* Acesse o link
* Clique em compartilhar
* "Adicionar à Tela de Início"

---

## ⚠️ Observações

* O projeto foi desenvolvido com foco em aprendizado e portfólio
* Os dados são armazenados localmente no navegador (localStorage)
